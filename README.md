## Fn: plot()

option: `style` values:

- plot.style_line (default)
- plot.style_stepline
- plot.style_stepline_diamond
- plot.style_histogram
- plot.style_cross
- plot.style_area
- plot.style_columns
- plot.style_circles
- plot.style_linebr
- plot.style_areabr
- plot.style_steplinebr

linestyle (input plot_line_style)

- plot.linestyle_solid - ciągła
- plot.linestyle_dashed - przerywana

- hline.style_solid (default)
